# Zama Confidential Token SDK

TypeScript SDKs for privacy-preserving ERC-20 token operations using [Fully Homomorphic Encryption](https://www.zama.ai/fhevm) (Zama fhEVM). Shield, transfer, and unshield tokens with encrypted balances — no one sees your amounts on-chain.

## Packages

- **[@zama-fhe/sdk](guides/sdk/overview.md)** — Core SDK: confidential token operations, FHE relayer, contract call builders, viem/ethers adapters, Node.js worker pool
- **[@zama-fhe/react-sdk](guides/react-sdk/overview.md)** — React hooks wrapping the core SDK via `@tanstack/react-query`, with viem/ethers/wagmi sub-paths

Get started with the [Quick Start](guides/quick-start.md) guide.
