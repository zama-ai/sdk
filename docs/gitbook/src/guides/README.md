# Overview

**Welcome to Guides!**

Step-by-step guides for common tasks with the Zama SDK. Each guide covers a single topic with code examples for both the core SDK and React hooks.

## Where to go next

If you're new to the Zama Protocol, start with the [Litepaper](https://docs.zama.ai/protocol/zama-protocol-litepaper) or the [Protocol Overview](https://docs.zama.ai/protocol) to understand the foundations.

Otherwise:

🟨 Go to [**Configuration**](configuration.md) to set up the relayer, signer, and storage.

🟨 Go to [**Authentication**](authentication.md) to learn about API key management and backend proxies.

🟨 Go to [**Shield tokens**](shield-tokens.md) to convert public ERC-20 tokens into confidential form.

🟨 Go to [**Transfer privately**](transfer-privately.md) to send encrypted amounts on-chain.

🟨 Go to [**Unshield tokens**](unshield-tokens.md) to withdraw confidential tokens back to public ERC-20.

🟨 Go to [**Check balances**](check-balances.md) to decrypt and display confidential balances.

🟨 Go to [**Decrypt UX lifecycle**](decrypt-ux-lifecycle.md) to avoid blind-sign wallet popups when displaying confidential balances.

🟨 Go to [**Handle errors**](handle-errors.md) to manage SDK errors gracefully in your app.

🟨 Go to [**Node.js backend**](node-js-backend.md) to run the SDK server-side with worker threads.

🟨 Go to [**Encrypt & decrypt**](encrypt-decrypt.md) for low-level FHE encryption and decryption operations.
