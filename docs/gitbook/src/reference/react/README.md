# Overview

**Welcome to the React reference!**

API reference for the `@zama-fhe/react-sdk` package. Each page documents a single hook or component with parameters, return values, and working code examples. All hooks are built on TanStack Query with automatic cache invalidation and cached decryption.

## Where to go next

🟨 Go to [**ZamaProvider**](ZamaProvider.md) for the required context provider that wires up the relayer, signer, and storage.

🟨 Go to [**useConfidentialBalance**](useConfidentialBalance.md) to decrypt and display a single token's balance.

🟨 Go to [**useShield**](useShield.md) to convert public ERC-20 tokens into confidential form.

🟨 Go to [**useConfidentialTransfer**](useConfidentialTransfer.md) to send encrypted amounts on-chain.

🟨 Go to [**useUnshield**](useUnshield.md) to withdraw confidential tokens back to public ERC-20.

🟨 Go to [**Query keys**](query-keys.md) for manual cache invalidation and custom query composition.
